HSK Flashcards

HSK Flashcards is a web-based React app for learning Chinese vocabulary from HSK levels 1–3. Users can browse, shuffle, and star flashcards to customize their study sessions.

Features

Browse vocabulary by HSK Level (1–3).
Flip cards to view Pinyin, Hanzi, and English.
Star favorite cards for focused study.
Shuffle cards or reset order.
Track progress with a progress bar.
Toggle show/hide Pinyin.

Tech Stack

React 18 (via CDN)
Babel for JSX transformation
HTML, CSS for UI

Usage

Open index.html in a browser.
Select HSK level and study mode (Browse/Starred).
Click cards to flip and learn.
Star important cards for review.
Navigate using Back/Next buttons.
